>What is this project

This project is a blog, where you can:
1. read/create/search for/comment under/delete your own/like/dislike posts
2. create an account and log in
3. follow other blog writers

Additionally the moderators of the website can:
1. delete other posts/comments
2. ban other accounts from posting/commenting
3. pin comments at the top

Optional goals:
1. notify a user when a writer they follow has posted/edited/got banned
2. if a user gets banned all of their posts become hidden untill a moderator approves of them
3. email verification on account creation

Website layout and Mechanics
1. the main page, shows all blogs, can sort by most liked/most disliked/newest/oldest
2. the follow page, the same as main page, but only shows the posts of the creators you follow
3. the account page, gives an option to create/delete/log out of an account
